Well, this is not that hard as it could be, buts it’s a bit expensive: I had to buy a VPS for that to proxy everything from https://defman.me to https://defman21.github.io (there’s some CNAME magic and I don’t want to dig in it, it just works). But I really wanted HTTPS on my web-site so I think it’s worth it.
