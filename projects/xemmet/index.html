Xemmet
the essential toolkit for web-developers using Komodo
Features
    Expand Abbreviation
    You can easily expand .block>ul>li{Item $$}*4 to 
    <div class="block">
    <ul>
        <li>Item 01</li>
        <li>Item 02</li>
        <li>Item 03</li>
        <li>Item 04</li>
    </ul>
</div>
    To do that, put your caret after Emmet abbreviation and press Tab.
    Wrap Selection
    Wrap Selection is a re-work of original Emmet feature. It's using your editor instead of a prompt window. You can select any text, press Ctrl+Tab and type an abbreviation you want to wrap your text with.
    For example, if you type .div>.article>.title, your text will be wrapped with: 
    <div class="div">
    <div class="article">
        <div class="title">{Your text}</div>
    </div>
</div>
    Xemmet looks for {} in your abbreviation and it will put your text at this place. If there's no such brackets, Xemmet will automatically add them to your abbreviation.
    Toolbox Integration
    Unlike in Emmet, where you have to define your snippets in a special file, Xemmet provides a very simple way to define them. You can create a snippet that Xemmet will recognize right in your Toolbox.
    The snippet should starts with xemmet_ in its name and be in Abbreviations folder. I'd recommend to create html and css folders under Abbreviations for faster search and easier control over your snippets. You can override default snippets of course. You can view the default snippets in content/extra/snippets.js.
    This is disabled by default because this could cause some lags. It works fine for me, but my PC is pretty strong.
    Flexible Settings
    Xemmet provides different settings to customize when and how it should works. You can find them under Editor - Smart Editing. These settings are pretty understandable except for Search for built-in snippets recursively. In Xemmet, some of the default snippets refers to another. For example, menuc refers to menucontext snippet. If the checkbox is not active, menucontext will be expanded to a HTML element which is kinda invalid. This checkbox will enable recursive searching in snippets to avoid this.
